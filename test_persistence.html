SpineLift Persistence Test

This test will verify that projects persist across page refreshes.

Instructions:

  1. Open the application at http://localhost:5173
  2. Log in with your credentials
  3. Upload a PSD file and wait for processing
  4. Once layers are shown, refresh the page (F5)
  5. You should see:
    • ✅ User stays logged in
    • ✅ Project is still loaded
    • ✅ Layers are displayed
    • ✅ Dark theme UI with 3 columns

  • Center panel (flexible): Mesh preview canvas
